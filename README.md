# SafeMax — неофициальная модификация мессенджера MAX

SafeMax — неофициальная модификация популярного мессенджера MAX с расширенными возможностями для опытных пользователей.

---

## Возможности

### 🤨 Старые корректировки
- Удалены трекеры и аналитика
- Вырезаны лишние разрешения
- В разделе помощь можно открыть Яндекс
- Добавлено антиудаление сообщений
- 🔓 Открыто меню разработчика
- 📞 Рабочие звонки
- ❤️ Удалена инициализация MyTracker
- 🔸 Некоторые запросы переадресованы на 0.0.0.0

### 🏬 Новые корректировки
- 😊 Вырезано обращение к Telegram (`main.telegram.org`), WhatsApp (`mmg.whatsapp.net`) и к сервисам, определяющим IP
- ⚙️ Добавлена подмена устройства (устройство на котором выполнен вход будет определяться как Samsung)
- 😎 Режим призрака
- 🤔 Отключен SSL Pinning

3. Установите и запустите
